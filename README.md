## 소감 및 건의사항
> * 인텔리제이라는 프로그램을 이번에 처음 써보게 되었는데 UI가 직관적이고 도구가 유용하여 쓰는데 불편함을 느끼지 못했습니다. 저번학기에 오픈 소스 소프트웨어 실습을 들으며 깃을 다루는 방법을 배워둔 덕분에 깃허브와 연동하여 쓰는데도 큰 어려움이 없었습니다.    
> * 다만, 자바 공부를 조금 더 많이 해야함을 느꼈습니다. 솔직히 말해서 강의동영상이 없었다면 수행하는데 꽤 애먹었을거라고 생각합니다. 강의동영상을 보며 따라하다보니 점점 자바 문법을 어떻게 써야 했는지 기억이 떠올랐고, 반복되는 항목이 많아서 후반에 들어서면서는 동영상 없이도 코딩하였습니다.    
> * 간단한 프로그램이었지만, 프로젝트 하나를 무사히 끝내게 되어 기쁩니다. 또한 적지않은 성취감도 보상으로 받을 수 있었습니다.
